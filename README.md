# chart.css

> v1.0.0

A Simple CSS Chart System

## How To Use

### 1. Include chart.css file

```html
<link rel="stylesheet" href="chart.css">
```

### 2. Set up HTML

```html
<div class="charts">
  <div class="charts__chart"></div>
  ...
</div>
```
